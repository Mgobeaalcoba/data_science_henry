# Clase 01: Introducción al Machine Learning

## Objetivos
- Comprender los conceptos fundamentales del Machine Learning
- Conocer los diferentes tipos de aprendizaje
- Entender el pipeline de un proyecto de ML
- Introducción a la preparación de datos

## Contenidos

### 1. ¿Qué es Machine Learning?
- Definición y conceptos clave
- Diferencia entre ML, IA y Deep Learning
- Aplicaciones del mundo real

### 2. Tipos de Aprendizaje
- **Aprendizaje Supervisado**: Clasificación y Regresión
- **Aprendizaje No Supervisado**: Clustering y Reducción de dimensionalidad
- **Aprendizaje por Refuerzo**

### 3. Pipeline de un Proyecto ML
1. Definición del problema
2. Recolección de datos
3. Exploración y análisis (EDA)
4. Preparación de datos
5. Feature Engineering
6. Selección de modelo
7. Entrenamiento
8. Evaluación
9. Deployment

### 4. Preparación de Datos
- Limpieza de datos
- Manejo de valores faltantes
- Encoding de variables categóricas
- Escalado de features
- Train/Test split

## Notebooks

1. **01_introduccion_conceptos.ipynb**: Conceptos básicos y terminología
2. **02_tipos_aprendizaje.ipynb**: Ejemplos de cada tipo de aprendizaje
3. **03_preparacion_datos.ipynb**: Técnicas de preparación de datos
