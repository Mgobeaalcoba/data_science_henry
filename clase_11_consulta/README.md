# Clase 11: Consulta Final

## Objetivos
- Repaso de conceptos clave del curso
- Resolución de dudas
- Discusión de proyectos finales
- Q&A abierto

## Formato de la Clase

### 1. Repaso Rápido (30 min)
Revisión de los temas principales:
- Machine Learning básico
- Aprendizaje supervisado (Regresión y Clasificación)
- Modelos de ensamble
- Optimización de modelos
- Aprendizaje no supervisado
- Series temporales
- Deep Learning

### 2. Resolución de Dudas (60 min)
- Preguntas conceptuales
- Dudas sobre implementaciones
- Problemas en proyectos
- Mejores prácticas

### 3. Proyectos Finales (30 min)
- Presentación de proyectos
- Feedback y sugerencias
- Tips para portfolio

### 4. Q&A Abierto (30 min)
- Temas avanzados
- Carrera en Data Science
- Recursos para continuar aprendiendo

### Carrera
